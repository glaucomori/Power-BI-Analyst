# Modelagem e Transformação de dados com DAX com Power BI

# Carga

- Todos dados usados originam da mesma base "Financial Sample.xlsx".
- A tabela FATO contém o objetivo da modelagem: os dados referentes às vendas.
- As tabelas de DIMENSÃO consistem dos detalhamentos das informações da tabela FATO.

# Transformação

- Para compor as tabelas foi necessário aplicar agrupamento de colunas usando funções de média, mediana, soma, mínimo e máximo.
- Também foram criadas colunas condicionais para transformar os dados para maior coerência para os relacionamentos das tabelas e para a futura visualização das informações.
